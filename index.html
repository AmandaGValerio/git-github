Curso de Git e GitHub

Este site foi elaborado para os alunos do curso de Git e GitHub ofertado no VI EnComp no ano de 2020. O material aqui disponibilizado foi ensinado durante todo o curso, além de ser apenas o básico para iniciar no mundo de Git e GitHub. Estarei atualizando este site com alguns links para aprofundar mais o seu conhecimento além de referências utilizadas para todo o material. Bons estudos!

Configurando o Git

git config --global user.name "Seu nome"

git config --global user.email "exemplo@seuemail.com.br"

Criando uma pasta

mkdir nomeDaPasta

Inicializando

git init // Lembre-se de estar dentro da pasta criada

Ver se houve alterações na pasta para novo ponto na história

git status

// Caso aparecer em vermelho ainda não foi adicionado no histórico

// Caso aparecer em verde já foi adicionado no histórico

// Caso contrário, significa que não tem nada alterado ainda

Criar arquivo

git code nomeArquivo.extensao // Ex.: txt, html, css, md...

Adicionar alterações no ponto da história

git add nomeArquivo.extensao // Adiciona o arquivo para o próximo ponto do histórico

git add . // Adiciona todas as alterações para o próximo ponto do histórico

Fazer um commit e acessar o histórico

git commit -m "Mensagem de sua preferência" // Marcar ponto no histórico

git log // Acessa o histórico

Push e Pull

git push -u origin master

// Envia para o repositório remoto origin na branch master os commits do seu repositório local

git pull origin

// Traz para seu repositório local os commits do repositório remoto origin

Branch(es)

git branch nomeDaBranch // Cria a branch

git branch // Mostra as branches existentes

git pull origin

git checkout nomeDaBranch // Vai para a branch selecionada

git remote / Mostra os repositórios remotos existentes

Fork

git clone linkDoSeuFork // Clona o seu fork

git remote add nomeDoSeuRemote linkDoOriginal // Adiciona um remote para o repositório original

git fetch nomeDoSeuRemote // Atualiza o remote

git rebase nomeDoSeuRemote/master // Atualiza o repositório local

Manter Fork atualizado

git fetch nomeDoSeuRemote

git rebase nomeDoSeuRemote/master

git push origin master
